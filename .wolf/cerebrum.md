## Decision Log

<!-- Significant technical decisions with rationale. Why X was chosen over Y. -->

- [2026-06-17] **ESP kararı: Brevo İPTAL → Listmonk (self-host) + Amazon SES (SMTP relay), eu-central-1 (Frankfurt).** Sebep: KVKK için veri kontrolü (abone listesi kendi DB'mizde, AB bölgesi), maliyet (SES ~$0.10/1000 e-posta, Listmonk ücretsiz), spam/teslimat (kendi domain DKIM/SPF/DMARC + SES IP itibarı). Brevo'nun barındırılan listesi + fiyatlandırması yerine. SES DNS (DKIM 3xCNAME + custom MAIL FROM `mail.stokoloji.com` MX+SPF + DMARC) girildi. **Kalan panel işi: SES sandbox'tan çık (production access) + SMTP credential üret; Listmonk panelinde Settings>SMTP (`email-smtp.eu-central-1.amazonaws.com:587` STARTTLS) + API user/token + liste (double opt-in).** Eski Brevo memory'si güncellendi.
- [2026-06-17] **Formül dizgisi = KaTeX (server-side), convention: code-fence içeriği.** Eski hâl: formüller markdown code-fence → Strapi `code`/inline `code` → düz gri monospace `√L`/`σ_d` ASCII taklit (otorite sitesine yakışmıyor). Karar: KaTeX, **`katex.renderToString` ile SSR/build'de HTML** (client JS yok → CWV korunur, kural 7). `BlocksRenderer` `code` bloğu kuralı: **backslash içeriyorsa gerçek LaTeX → KaTeX display**; düz metinse (kelimeli Türkçe formül) → şık metin kartı (`.formula-block--text`), KaTeX DEĞİL. Sebep: KaTeX matematik fontu (Computer Modern) Türkçe glifte (ğ/ş/ı dotless) zayıf, `\text{Türkçe}` tofu riski → kelimeli formüller (ROP "Ortalama günlük talep × ...") text kalır, sembolik olanlar (σ/√/subscript) LaTeX olur. Label için `SS` kısaltması (içerikte zaten kullanılıyor) — `\text{Emniyet Stoğu}` glif riskinden kaçınmak için. Satır-içi `$...$` → KaTeX inline. Türkçe ondalık virgül LaTeX'te `1{,}65` (sıkı aralık). Launch-icerik emniyet+EOQ fence'leri LaTeX'e çevrildi; ROP kelimeli kaldı.

- [2026-06-16] **SEMrush (tr) düzeltmesi:** "stok yönetimi" head term'i ulaşılabilir — hacim 320, **KD 17 (çok kolay)**; "ERP devleri domine, girilemez" varsayımı yanlıştı (markalar ticari intent sayfalarına yatırım yapıyor, bilgi sorgusunda zayıf).
